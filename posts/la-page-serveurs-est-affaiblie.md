La page serveurs de MultiCraft Info est un pilier principal du site.
Avec plus de 2700 serveurs indexés, nous sommes la plus grande base de serveurs MultiCraft existante.

Récemment, il s'avère que MultiCraft à modifié son API, il nous est désormais impossible de récupérer les serveurs, le nombre de joueurs, la description des serveurs etc. jusqu'à ce que nous trouvions une correction, ce qui risque d'être très long.

Concrètement, cela signifie :
- Nous ne pouvons plus récupérer les nouveaux serveurs. 
- Nous ne pouvons plus récupérer la description, le nom et le nombre de joueurs des serveurs.
- En revanche, la liste des joueurs connectés fonctionne encore pour le moment.

Cela rend notre page Serveurs beacoup moins riche qu'avant.

Cependant, nous avons mis en place une API temporaire pour le nombre de joueurs. Concrètement, cette API demande à MultiCraft la liste des joueurs et calcule par la le nombre de joueurs du serveur.
Cette API est peu fonctionnelle, prend environ 1 heure et demi pour mettre à jour l'entièreté des serveurs, et risque d'être détectée et bloquée assez rapidement par MultiCraft.

Sachez que l'équipe est actuellement pleinement mobilisée à rétablir un fonctionnement normal au plus vite. Nous ne pouvons pas vous donner de date.

- L'équipe MultiCraft Info
